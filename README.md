# 🌐 Network

> Batch script for **network registry and service configuration** — SetACL on Network key, devcon, backup permissions, DHCP, DUSM, NLA, etc.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **SetACL** | Removes Everyone, sets Administrators/SYSTEM/Users on Network key |
| 🔧 **devcon** | Device driver control (if present) |
| 📝 **Logging** | `network_cleanup_log.txt` |
| 🔒 **Ownership** | Sets Administrators as owner |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 |
| **SetACL.exe** | In script directory |
| **Privileges** | Administrator |
